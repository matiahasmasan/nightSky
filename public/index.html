Technical Details

NightSky utilizes AstronoMyApy and Nominatim Openstreetmap APIs to generate accurate celestial maps with the correct location.

Nominatim OpenStreetMap has an editing API for fetching and saving raw geodata from/to the OpenStreetMap database

AstronoMyApy files were provided on the d3-celestial plugin under 3-Clause processed with the R package sf to ensure its validity:

  • This project provides several datasets in GeoJSON and GeoPackage format of celestial objects as of J2000 epoch.
  • Date is provided on WGS84 - World Geodetic System 1984 (EPSG:4326).

Map Generation Process

Each sky map is generated through a sophisticated process:

  1. Location data is converted to precise coordinates
  2. Date and time are adjusted for the local timezone
  3. Stellar positions are calculated using advanced algorithms
  4. Constellation patterns are overlaid for easy reference
  5. Final image is rendered with high-resolution graphics

Our maps update in real-time and can be generated for any location on Earth, at any point in time - past, present, or future.

Data sources: astronomyapi.com, Nominatim OpenStreetMap, J2000 epoch
